Terminal Commands

  • pwd
    • (Print Working Directory)
  • ls -a
  • cd folder
    • Puts you in the specified folder
  • cd documents/
    • Puts you in the "Documents" folder
  • cd /z
    • Puts you in the "c" drive
  • cd ..
    • Sends you back
  • mkdir folder name
    • Makes a new directory/folder
  • ~/ user folder (aka /c/users/username)
  • / root
  • ./ current directory
  • ../ enclosing directory
  • touch file.txt
    • Makes a new file
  • which command
  • nano file.txt
  • cat file.txt
    • Prints the text
  • mv /folder name/file.txt ./
    • Moves a file from the origin to a new destination
  • mv /file.txt/newfile.txt
  • mv /file.txt/newname.js
    • Renames a file and/or its file extension
  • cp /newname.js/newfile.txt
    • Renames a file and/or its file extension
